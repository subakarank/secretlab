## Routing
 Store and update object - POST method
- /object 
```{ "keName": "value"}```

 Get object - GET method
 - /object/{keyName}

 Get object with timestamp - GET method
 - /object/{keyName}?timestamp={unixTimestamp}

 Get all records  - GET method
  - /object/get_all_records
